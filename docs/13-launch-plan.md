## Pre-Launch (4 weeks before)

### Week -4: Build & Polish
```
├── Complete MVP feature set
├── Fix critical bugs from internal testing
├── Set up analytics (Posthog)
├── Set up error tracking (Sentry)
├── Set up Stripe billing (test mode)
└── Write launch copy
```

### Week -3: Beta Users
```
Target: 20 beta users

Who to invite:
├── 5 VCs / angel investors (LinkedIn connections)
├── 5 strategy consultants (Twitter/X network)
├── 5 startup founders (communities you're in)
└── 5 journalists / analysts (cold outreach)

What to ask from beta users:
├── Run 5+ research tasks
├── 20-minute feedback call
├── Rate report quality 1-10
└── "Would you pay for this? How much?"

Success criteria:
├── >70% rate reports 7+/10
├── >50% say they'd pay
└── Clear signal on what to fix before launch
```

### Week -2: Fix Beta Feedback
```
├── Address top 3-5 issues from beta
├── Improve report quality based on feedback
├── Refine onboarding flow
├── Set up customer support (Intercom or Crisp)
└── Write FAQ / help docs
```

### Week -1: Launch Prep
```
├── Prepare Product Hunt listing:
│   ├── Tagline: "One question. Ten agents. Complete picture."
│   ├── Screenshots (5): input, agents running, report, export, pricing
│   ├── Demo GIF (30 seconds): question → agents → report

├── Write HN Show HN post
├── Prepare Twitter/X thread
```
